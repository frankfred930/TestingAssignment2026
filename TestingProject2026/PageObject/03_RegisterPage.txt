# Positive Tests

Register user with valid username and password
    Navigate To Registration Page    LoginUrl
    Input Username    Valid_Username
    Input Email    Valid_Email
    Input Password    Valid_Password
    Confirm Password    Valid_Password
    Complete Recaptcha    (Manual step - recaptcha cannot be automated)
    Click Register Button
    Verify User Is Redirected To Welcome Page Or Onboarding Page
    # Remember credentials which you are going to use for whole application testing

# Negative Tests

Register user with same existing username
    Navigate To Registration Page    LoginUrl
    Input Username    Existing_Username
    Input Email    newemail@example.com
    Input Password    Valid_Password
    Confirm Password    Valid_Password
    Click Register Button
    Verify Error Message "Username has already been taken" Is Displayed

Register user with empty password
    Navigate To Registration Page    LoginUrl
    Input Username    new_test_user
    Input Email    newuser@example.com
    Input Password    Empty
    Confirm Password    Empty
    Click Register Button
    Verify Error Message "Password can't be blank" Is Displayed

Register user with invalid email format
    Navigate To Registration Page    LoginUrl
    Input Username    user_with_bad_email
    Input Email    Invalid_Email
    Input Password    Valid_Password
    Confirm Password    Valid_Password
    Click Register Button
    Verify Error Message "Email is invalid" Or "Please provide a valid email address" Is Displayed

Register user with password too short (minimum 8 characters required)
